# emorobcare.github.io

Robot Video Web

Simple demo site: abre `index.html` en un navegador para ver el título centrado.

Ver localmente

- Doble clic en `open-index.bat` para abrir la página en tu navegador por defecto.
- Ejecuta en PowerShell para abrir directamente con políticas temporales:

```
powershell -ExecutionPolicy Bypass -File open-index.ps1
```

- Para servir en `http://localhost:8000` (requiere Python):

```
powershell -ExecutionPolicy Bypass -File serve.ps1
```
